## Download our object

To download our cheesecake photo, let's use the copy command:

{% code-group %}
```windows
./uplink.exe cp sj://cakes/cheesecake.jpg ~/Downloads/cheesecake.jpg
```

```macos
uplink cp sj://cakes/cheesecake.jpg ~/Downloads/cheesecake.jpg
```

```linux
uplink cp sj://cakes/cheesecake.jpg ~/Downloads/cheesecake.jpg
```
{% /code-group %}
