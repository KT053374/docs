## Delete our object

Let's delete our photo with the following command:

{% code-group %}

```windows
./uplink.exe rm sj://cakes/cheesecake.jpg
```

```macos
uplink rm sj://cakes/cheesecake.jpg
```

```linux
uplink rm sj://cakes/cheesecake.jpg
```

{% /code-group %}
